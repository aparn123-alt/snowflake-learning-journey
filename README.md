# Snowflake Learning Journey

## Week 1 — What stood out to me

- Snowflake separates storage and compute — my data is always saved, but I only pay for compute while it's actively running a query, thanks to auto-suspend.
- Object hierarchy is simple: Account → Database → Schema → Tables.
- RBAC means permissions go to roles, not individual users — easier to manage access at scale.
- Time Travel was the standout feature — recovering deleted/changed data using AT/BEFORE, no manual backup needed.
- Did all of this hands-on in a live trial account — real queries, real mistakes, real recovery.
